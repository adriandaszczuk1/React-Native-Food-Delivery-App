## Setup

```bash
yarn
```

or

```bash
npm i
```

## Running

* iOS:
  * Yarn: `yarn ios`
  * npm: `npm run ios`

* Android:
  * Yarn: `yarn android`
  * npm: `npm run android`

* Later builds:
  * Yarn: `yarn start`
  * npm: `npm run start`
